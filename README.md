## Current status

Since the devs have added obfuscation to their code after becoming aware of this mod, the code to modify game files and generate patches has been removed from the repository for now.

The patches themselves are still available.

I'm keeping them up because I see this mod as helping others. If you need it, you know. If you want it, you know. If you don't need or want it, that's completely fine - it won't affect your game or give anyone else any advantage.

This mod only changes displayed text. No gameplay, no balance, nothing under the hood.

I may revisit this later depending on how things evolve.
